# User Manual

## Getting started
Open your browser and go to http://localhost:3001

## Classifying a review
1. Paste or type a product review into the text box
2. Select the aspect you want to analyse (price, quality, delivery, service)
3. Click Classify
4. The result appears below the form showing:
   - Sentiment: positive, neutral, or negative
   - Confidence: how certain the model is (0-100%)
   - A yellow warning if confidence is below 60% — verify these manually

## Reading the dashboard

- The bar chart breaks down sentiment per aspect
